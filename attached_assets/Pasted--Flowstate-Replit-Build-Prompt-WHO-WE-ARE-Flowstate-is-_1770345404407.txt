# Flowstate â€” Replit Build Prompt

## WHO WE ARE

Flowstate is a lifestyle brand disguised as a campus health app. Built by Tate and Maya. The core philosophy: **when you nail the essentials â€” eating right, moving your body, resting well â€” you unlock your own flow state.** The app is the tool. The brand is the identity. People should want to wear Flowstate on a hat before they even download the app.

This is NOT a generic health directory. It's a system for locking in. The vibe is: confident, kinetic, a little competitive, never preachy. Think Nike's mindset meets a college student's daily reality.

**The metaphor that drives everything:** In a real flow state, noises dim and your eyes focus harder. The app should literally reflect this. When you open the app, the world is blurry â€” as you lock in and hit your essentials, everything sharpens into focus. The UI itself tells the story of entering flow.

## EXISTING TECH STACK (keep this the same)

- React 18.2.0
- Vite 4.3.9 (with Terser minification)
- Lucide React icons
- Custom CSS (no Tailwind, no UI framework)
- CSS variables for dynamic school theming

### FoodScannerModal.jsx
```jsx
import React, { useState } from 'react';
import { Camera, X, Zap, DollarSign, ScanLine } from 'lucide-react';

const FoodScannerModal = ({ onClose, schoolColors }) => {
  const [scanned, setScanned] = useState(false);
  const [scanResult, setScanResult] = useState(null);

  const handleScan = () => {
    setTimeout(() => {
      setScanResult({
        name: 'Organic Quinoa Bowl',
        calories: 420,
        protein: '18g',
        healthScore: 92,
        ingredients: ['Quinoa', 'Avocado', 'Black Beans', 'Kale', 'Lime'],
        priceComparison: [
          { store: 'Whole Foods', price: '$8.99', distance: '1.8 mi' },
          { store: 'Publix', price: '$7.49', distance: '0.9 mi' },
          { store: 'Target', price: '$6.99', distance: '2.3 mi' }
        ]
      });
      setScanned(true);
    }, 2000);
  };

  return (
    <div className="modal-overlay">
      <div className="modal-content">
        <button onClick={onClose} className="modal-close"><X size={20} /></button>
        <div className="modal-body">
          {!scanned ? (
            <>
              <div className="scanner-header">
                <div className="scanner-icon"><Camera size={40} color="white" /></div>
                <h2 className="scanner-title">Scan Your Food</h2>
                <p className="scanner-subtitle">Get instant nutrition info and price comparisons</p>
              </div>
              <div className="camera-placeholder">
                <div className="scan-frame"><ScanLine size={48} color={schoolColors.primary} /></div>
              </div>
              <button onClick={handleScan} className="btn-primary full-width">Start Scanning</button>
            </>
          ) : (
            <>
              <div className="results-header">
                <div className="success-icon"><Zap size={30} color="white" /></div>
                <h3 className="result-name">{scanResult.name}</h3>
                <div className="health-score">{scanResult.healthScore}/100</div>
                <p className="score-label">Health Score</p>
              </div>
              <div className="nutrition-box">
                <h4 className="nutrition-title">Nutrition Facts</h4>
                <div className="nutrition-grid">
                  <div><div className="nutrition-value">{scanResult.calories}</div><div className="nutrition-label">Calories</div></div>
                  <div><div className="nutrition-value protein">{scanResult.protein}</div><div className="nutrition-label">Protein</div></div>
                </div>
              </div>
              <div className="price-section">
                <h4 className="price-title"><DollarSign size={16} />Price Comparison</h4>
                {scanResult.priceComparison.map((store, i) => (
                  <div key={i} className={`price-row ${i === 0 ? 'best-price' : ''}`}>
                    <div><div className="store-name">{store.store}</div><div className="store-distance">{store.distance} away</div></div>
                    <div className="store-price">{store.price}</div>
                  </div>
                ))}
              </div>
              <button onClick={() => { setScanned(false); setScanResult(null); }} className="btn-secondary full-width">Scan Another Item</button>
            </>
          )}
        </div>
      </div>
    </div>
  );
};

export default FoodScannerModal;
```

## LOGO DESIGN SPEC

Design an SVG logo for Flowstate that will be used throughout the app (welcome screen, header, favicon) and also needs to work embroidered on hats and printed on merch.

### Logo Concept: The Flow Mark

The logo should be a **custom wordmark + abstract icon mark** combination.

**The Icon Mark (The "Flow" Symbol):**
- An abstract, continuous line or wave form that conveys fluid motion and momentum â€” the feeling of being "in the zone"
- Think of it as a stylized letter "F" that flows into an infinite/continuous shape, or an abstract wave that implies forward movement
- It should feel KINETIC â€” like it's in motion even when static
- The shape should be simple enough to work as: a 1-inch hat embroidery, an app favicon at 32x32px, a single-color sticker
- Avoid: shields, health crosses, leaves, generic wellness symbols, anything that screams "health app"
- The mark should feel like a lifestyle/athletic brand â€” closer to Nike swoosh energy than a medical logo

**The Wordmark:**
- "Flowstate" or "FLOWSTATE" in a clean, bold, modern sans-serif
- Slightly custom letterforms â€” maybe the "F" connects to the icon mark, or the letters have subtle flow/momentum to them
- Should feel premium and confident, not playful or bubbly

**Color Versions:**
- Full color: Uses the school's primary and secondary colors (default: deep navy #0A3D62 and green #00A651)
- Single color: Works in all-white (for dark backgrounds, hat embroidery on dark caps)
- Single color: Works in all-black (for light backgrounds)
- The logo must be legible and impactful in single-color applications

**Where it appears in the app:**
- Welcome page: Large, centered, above the subtitle (replaces the current plain text h1)
- Dashboard header: Small version, left-aligned (replaces current "Flowstate" text)
- Initial loader screen: Animated version with the blur-to-focus effect (replaces current pulsing text)
- Favicon: Icon mark only

### SVG Implementation:
- Create the logo as an inline SVG React component called `FlowstateLogo`
- Props: `size` (scales proportionally), `color` (for single-color mode), `variant` ('full' | 'icon' | 'wordmark')
- Use `currentColor` for single-color flexibility
- Keep the SVG path count minimal for performance

## DESIGN PRINCIPLES

6. **Works on a hat.** If a visual element can't be reduced to a simple mark, it doesn't belong in the brand.
