## Getting Started on Replit

1. **Create a new Repl** and select "Import from GitHub" or create a blank React Repl

2. **Upload these files:**
   - `package.json`
   - `vite.config.js`
   - `index.html`
   - `.replit`
   - `src/main.jsx`
   - `src/flowstate-app.jsx`

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open the app** in your browser at the Replit URL

## How to Use

1. **Sign Up:** Enter your student email (must be @eagle.fgcu.edu)
2. **Select School:** Choose FGCU from the dropdown
3. **Set Location:** Enter your current location or use device location
4. **Explore:** Browse healthy eating spots, gyms, and grocery stores
5. **Scan Food:** Use the food scanner to check nutrition and prices

## Tech Stack

- **React** - UI framework
- **Lucide React** - Icon library
- **Vite** - Build tool
- **Custom CSS** - Styling with FGCU brand colors

## Customization

To add more schools, update the `SCHOOL_COLORS` object in `flowstate-app.jsx`:

```javascript
const SCHOOL_COLORS = {
  fgcu: {
    primary: '#0A3D62',
    secondary: '#00A651',
    accent: '#FFB81C',
    name: 'Florida Gulf Coast University',
    shortName: 'FGCU'
  },
  // Add more schools here
};
```

## Design Philosophy

Flowstate uses a vibrant, energetic design that appeals to college students while maintaining credibility and trustworthiness. The app features:

- **Bold typography** with Poppins for headlines and Inter for body text
- **Dynamic gradients** using school colors
- **Smooth animations** for a modern feel
- **Card-based layouts** for easy scanning
- **Mobile-first design** for on-the-go students
